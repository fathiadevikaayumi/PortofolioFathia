# Employee Salary System
Project ini adalah program sederhana untuk mengelola dan menghitung data gaji karyawan.

## Fitur
* Input data karyawan
* Perhitungan gaji
* Menampilkan data gaji

## Teknologi yang Digunakan
* PHP
* HTML
* CSS
* MySQL

## Tujuan Project
Project ini dibuat untuk memahami pengolahan data serta penerapan logika perhitungan dalam pemrograman.
