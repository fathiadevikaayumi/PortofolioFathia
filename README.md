# PortofolioFathia
Repository ini berisi beberapa project yang saya kerjakan selama belajar pemrograman di sekolah. Project-project ini dibuat untuk melatih logika pemrograman serta memahami pembuatan aplikasi berbasis web dan desktop.
## Daftar Project
### 1. Login System (PHP & MySQL)
Aplikasi web sederhana yang memiliki fitur login dan registrasi pengguna menggunakan PHP dan database MySQL.

### 2. Pacman Game (Python Tkinter)
Game Pacman sederhana yang dibuat menggunakan Python dengan tampilan GUI menggunakan Tkinter.

### 3. Employee Salary System
Program sederhana untuk mengelola data dan perhitungan gaji karyawan.

## Teknologi yang Digunakan
* Python
* PHP
* HTML
* CSS
* MySQL
* Tkinter
  
## Tujuan
Portfolio ini dibuat untuk mendokumentasikan hasil pembelajaran saya di bidang pemrograman serta mengembangkan kemampuan dalam membuat aplikasi sederhana.
