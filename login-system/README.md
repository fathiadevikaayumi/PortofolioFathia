# Login System (PHP & MySQL)
Project ini adalah aplikasi web sederhana yang memiliki fitur login dan registrasi pengguna.

## Fitur
* Registrasi pengguna
* Login menggunakan username dan password
* Validasi data pengguna
* Penyimpanan data ke database

## Teknologi yang Digunakan
* PHP
* HTML
* CSS
* MySQL

## Tujuan Project
Project ini dibuat untuk memahami konsep autentikasi pengguna serta penggunaan database dalam aplikasi web.
